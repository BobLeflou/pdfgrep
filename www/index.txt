== Overview

_Pdfgrep_ is a tool to search text in PDF files. It works similarly to
grep.

image:screenshot.png["scr"]

== Features

* search using regular expressions.
* support for some important grep options, including:
** filename output.
** page number output.
** optional case insensitivity.
** count occurrences.
** recursive search.
* and the most important feature: color output!

For a complete documentation, please consult the link:pdfgrep.html[manpage].

== Download

The latest version is 1.3.2. You can download it
link:https://sourceforge.net/projects/pdfgrep/files/[here].

The development happens on GitLab. You can view
link:https://gitlab.com/pdfgrep/pdfgrep[the code online] or simply
clone the git repository:

  git clone https://gitlab.com/pdfgrep/pdfgrep.git

== Contact

General questions, suggestions, bug reports, patches or anything else
can be sent to the mailinglist at
mailto:pdfgrep-users@lists.sourceforge.net[pdfgrep-users@lists.sourceforge.net].

You can also use the
link:https://gitlab.com/pdfgrep/pdfgrep/issues[issue tracker] for bug
reports or create a
link:https://gitlab.com/pdfgrep/pdfgrep/merge_requests[merge request]
on GitLab, if you prefer that over mailinglists.
